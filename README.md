technology stack

- Build Tool: Vite
- Styling: UnoCSS, Sass
- Programming Language: TypeScript
- Framework: Vue 3

Project File Structure and Descriptions

1. ./src/main.ts is the main entry point of the project.
2. ./src/style.css is the shared CSS file for the project.
3. ./src/App.vue is the entry component.
4. ./assets is the directory for storing static assets.
5. ./index.html is the root HTML file of the project.
6. ./unocss.config.ts is the configuration file for UnoCSS.
7. ./components/brand.vue is the component used for rendering the shortcut icon.
8. ./assets/logo.json stores the relationship between the company's domain names and their corresponding logo images.
9. ./package.json is the path of package.json
10. ./public/config.yaml is all user information, if you need to modify user information, please directly edit ./public/config.yaml.

Install

```shell
pnpm install
```

Build

```shell
Pnpm run build
```
